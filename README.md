Overview

This project analyzes a sales dataset using Python, Pandas, and Matplotlib.
It loads a CSV file, summarizes the data, and generates visualizations such as:

Monthly total sales (bar chart)

Product sales distribution (pie chart)

The tool helps users quickly understand sales performance trends.

Features

Load any CSV sales dataset

Show first 5 rows

Show summary statistics

Calculate total sales per month

Calculate total sales per product

Generate bar and pie charts
